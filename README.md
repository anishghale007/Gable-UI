# gable_ui

A UI clone of Gable (Job Portal HTML template). Link of the template: https://templates.hibootstrap.com/gable/default/index-2.html

NOTE: Landscape mode is not avaialble. Only POTRAIT mode is allowed. 

## Packages Used

- Get
- flutter_switch
- carousel_slider
- font_awesome_slider
